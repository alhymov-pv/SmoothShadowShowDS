# SmoothShadowShowDS
a test project for the dark background of a modal window in the form of an intermediate modal dark translucent window
DeepSeek prompt:
Delphi 6. Если нужно, сделай юнит с импортом GDI+(в Delphi 6 его, кажется, нет), с помощью которого будет работать следующий диалог. Диалог открывается размером и позицией с клиентскую область родительского окна. Назовём его "тень". В исходных файлах тени не следует использовать DFM. У тени нет ни рамки, ни заголовка, ничего, только фон. Фон заполняется чёрным градиентом от центра к краям. Форма градиента - подушкообразная, сферическая, конусная - задаётся в параметрах. Главное, что противоположный чёрному цвет - прозрачный. Т.е. по краям диалога будет просвечивать родительское окно. Также максимальная непрозрачность может быть ограничена. Т.е. через центр тоже будет видно родительское окно. По умолчанию, скажем, 50%. Тень показывается не сразу, а проявляясь из полностью прозрачного состояния за указанное время (секунды 2,5). Сразу при вызове тень должен показать другой диалог(назовём его "опции"), чтобы вернуть своим вызовом модальный результат опций. Пусть для примера опции - это будет диалог с галочкой, выпадающим списком и кнопками "ок", "отмена". Если опции закрыли раньше, чем время проявления тени, то проявление прекратить. После закрытия опций должно быть плавное исчезновение тени(где-то полсекунды). Для анимации использовать потоки. Не забыть, что делаем для Delphi 6 - ограничения синтаксиса и набора библиотек.
